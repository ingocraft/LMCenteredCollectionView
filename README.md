## Introduction

`LMCollectionView` is a infinite scroll view. 

## Installation

### CocoaPods

Add `LMCollectionView` in your `Podfile`.
```ruby
pod 'LMCollectionView'
```

Then,
```bash
pod install
```

### Manual

Copy `Sources` folder to your project. That's it.

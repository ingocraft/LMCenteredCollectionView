## Introduction

`LMCenteredCollectionView` is a infinite scroll view. 

## Installation

### CocoaPods

Add `LMCenteredCollectionView` in your `Podfile`.
```ruby
pod 'LMCenteredCollectionView'
```

Then,
```bash
pod install
```

### Manual

Copy `Sources` folder to your project. That's it.

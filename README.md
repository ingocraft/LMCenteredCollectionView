## Introduction

`LMdialView` is a infinite scroll view. 

## Installation

### CocoaPods

Add `InfiniteCollectionView` in your `Podfile`.
```ruby
pod 'InfiniteCollectionView'
```

Then,
```bash
pod install
```

### Manual

Copy `Sources` folder to your project. That's it.
